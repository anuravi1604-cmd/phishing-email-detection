# AI-Powered Phishing Email Detection System

A Machine Learning and NLP-based phishing email detection web application built using Python, Scikit-learn, and Streamlit. The system analyzes email/message text and classifies it as either phishing or legitimate.

---

## Live Demo

---

## GitHub Repository

https://github.com/anuravi1604-cmd/phishing-email-detection

---

## Features

- Real-time phishing email detection
- NLP-based text preprocessing
- TF-IDF vectorization
- Logistic Regression classification
- Probability-based phishing scoring
- Interactive Streamlit web interface
- Machine Learning model deployment
